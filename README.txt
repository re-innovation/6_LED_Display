This is the github repository for a 6 digit 7 segment LED board.
The board is based upon an ATmega328 microcontroller programmed with the Arduino Uno bootloader.
This project has been started by Matt Little from Renewable Energy Innovation.
Contact:
www.re-innovation.co.uk
matt@re-innovation.co.uk

Overview:

This kit contains 6 x 0.56" 7 segment LED units, along with an AS1107 SPI LED driver and Arduino microcontroller.

see www.re-innovation.co.uk for more details
